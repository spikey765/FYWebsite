Data Types
Data types are foundational elements in programming that define the nature and characteristics of data within a system. They specify the type of values that can be assigned to variables and dictate the operations that can be performed on them. Data types used in this course include (but not limited to) integers, strings, booleans and sets.

Integers

1, -1, 0, 1421, ...

An integer is a data type used to represent whole numbers without any fractional or decimal components. Integers can be positive, negative, or zero and are typically stored in memory using a fixed amount of space. They are fundamental for arithmetic operations, counting, indexing, and various computational tasks, serving as a basic building block for a wide range of applications and algorithms.

Booleans

True, False

In computer science, a boolean is a data type that represents two possible values: true or false. Booleans are used to make logical decisions, control the flow of programs, and evaluate conditions. They occupy a minimal amount of memory and play a crucial role in conditional statements, loops, and boolean algebra operations within algorithms and applications.

Characters

'a', '1', '!', ...

A character is a single unit of text or data that can represent a letter, number or symbol. Characters are the basic building blocks used to create strings, which are sequences of characters. Each character is typically encoded using a specific character encoding scheme, such as ASCII or Unicode, to represent it as a binary value that can be stored and manipulated by a computer.

Strings

"Hello World!", "123", "!", ...

In computer science, a string is a data type that represents a sequence of characters, often used to store and manipulate text-based information. Strings are fundamental in programming for tasks like input/output operations, text processing, and data representation. Individual characters can be referenced by treating a string as an array of characters.

Arrays

a = {1, 2}, b = {"ab", "cd"}, c = {}, ...

An array is a data structure that stores a collection of elements, each identified by an index. Arrays are typically used to store data of the same type, such as integers or characters, and provide efficient access to individual elements based on their position (index) in the array.

Sets

S = (1, 2, 3), B = ('a', 'b', 'c'), E = ∅, ...

In mathematics and computer science, a set is a collection of distinct elements without any specific order. Sets are used to group related items or data points and are characterized by their members, which can be numbers, objects, or other types of data. Operations like union, intersection, and difference can be performed on sets to combine, compare, or modify their contents.
